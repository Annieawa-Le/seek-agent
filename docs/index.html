核心能力

为开发者效率而生

每一项功能都旨在让你保持心流——从智能代码分析到无缝的 IDE 集成。

多模型 AI 支持

无缝集成 DeepSeek、OpenAI 及兼容模型。无需改变工作流即可热切换模型提供商。

DeepSeek OpenAI Custom

终端原生 TUI

完整的终端界面，支持 Markdown 渲染、语法高亮和键盘驱动交互。无 Electron 开销。

TUI ANSI 键盘

可扩展工具系统

动态工具注册、运行时热重载、补丁管理和子 Agent 编排。构建你自己的工具箱。

热重载 补丁 子 Agent

IDE 深度集成

VS Code 扩展支持内联代码编辑、文件系统感知，以及 Agent 与开发者之间的实时协作。

VS Code 内联编辑 文件感知

会话管理

持久化会话支持保存/恢复、内存压缩、上下文折叠，以及跨重启的自动状态管理。

持久化 压缩 恢复

技能 SDK

通过简洁的 SDK 创建和分享自定义技能。工具、提示词和工作流被打包为可组合的模块。

SDK 可组合 可分享
系统设计

它是如何工作的

清晰分层的架构，从终端界面到模型提供商和工具运行时，各层职责分明。

  • 01

    终端 UI 层

    事件驱动的 TUI，支持 Markdown 渲染、键盘快捷键和面板管理。

  • 02

    Agent 运行时

    消息循环含 Hook、内存管理、子 Agent 编排和错误恢复。

  • 03

    工具系统

    动态工具发现、基于补丁的编辑、参考桌面和热重载技能。
