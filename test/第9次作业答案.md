## 试题 4：计算机网络断言（哪个错误）
**答案：B**

- A ✓ AP 不是宿主机，可以实现有线和无线信号的相互转换。
- B ✗ **错误**。网络交换机连接运行相同协议的宿主机（同构网络），连接异构网络（如以太网和 WiFi）需要路由器。

## 试题 7：IPv6 对比 + 顶级域名
**答案：2^96；cn**

- IPv6 地址空间 = 2^128，IPv4 = 2^32
- IPv6 是 IPv4 的 2^128 / 2^32 = **2^96** 倍
- 网址 `https://cs101.gbu.edu.cn/` 的顶级域名是 **cn**（.cn 是中国国家顶级域名）

## 试题 12：搜索网络（哪个说法不正确）
**答案：D**

- A ✓ 搜索网络随时间变化（用户增减、搜索引擎变化），是演化网络。
- B ✓ 在任何时间点，网络拓扑是固定的，是静态网络。
- C ✓ 每个用户与搜索引擎相连，形成以搜索引擎为中心的星型结构。
- D ✗ **错误**。在任何时间点，网络拓扑是固定的（静态的），而非动态的。
