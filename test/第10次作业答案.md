# 第 10 次作业答案 — CS101（第 14 周）

## 试题 1：分组交换的正确说法
**答案：B**

- A ✗ 仅拆分成数据包，但不限于一个用户。
- B ✓ **将多个用户的多条消息拆分成数据包，依次随机发送。** 这是分组交换的核心特征——多用户共享信道，数据包独立路由。
- C ✗ 将多个用户消息打包成一个包传输，这实际上不是分组交换的做法。
- D ✗ 将一个用户的多条消息打包成一个包，也不是分组交换。

## 试题 2：为什么发明分组交换（多选）
**答案：ABCD**

所有选项都是合理的解释：

- A ✓ 分组交换中三个通信任务可同时进行，无需等待他人完成。
- B ✓ 电路交换中张三需要等待 7.31 秒才能开始传输，体验差。
- C ✓ 电路交换的信道容量未充分利用，分组交换通过共享更高效利用带宽。
- D ✓ 电路交换中李四需要等待 7.46 秒才能开始传输，体验差。

> 教材实例 6.20 说明：虽然总完成时间相同，但分组交换的并发性、无等待和利用率优势是发明它的根本原因。

## 试题 3：数据包的包头与包体
**答案：A**

## 试题 4：协议栈（哪个不正确）
**答案：A**

- A ✗ **不正确。** HTTP 消息的不同数据包可能经由不同物理路径传输（分组交换中每个包独立路由），不保证同路径。
- B ✓ HTTP 对等接口提供了抽象，使对等点无需关心下层协议。
- C ✓ 万维网 + 因特网的协议栈是 Web 通信的技术基础。
- D ✓ HTTP 和 TCP 之间的服务接口，由 TCP 层为 HTTP 层提供服务。

## 试题 5：协议栈（哪个不正确）
**答案：E**

- A ✓ HTTP 发送时，下层数据链路层必然有数据包发送。
- B ✓ 必然有 IP 数据包发送。
- C ✓ HTTP 基于 TCP，必然有 TCP 数据包发送。
- D ✓ 最终以物理层的二进制比特流传输。
- E ✗ **不正确。** HTTP 必须依赖 TCP、IP 和数据链路层，不能跳过它们直接传输。

## 试题 6：指数退避（Ethernet CSMA/CD）
**答案：2^N\*T**

表达式：**2^N\*T**

> 说明：第一次传输失败后等待区间为 [0, T]；此后每重试一次，区间上界翻倍。
> 第 N 次重试失败后（N ≥ 3），等待区间为 [0, 2^N × T]。

---

## 试题 7：最少路由器故障数
**答案：2**

路由器拓扑：核心层 (Z, X, Y, U, V) 全互联结构，T（北京）连接 X 和 U，W（上海）连接 Y 和 V。宿主机 A 通过 T 接入，B 通过 W 接入。

需要同时故障 2 个核心路由器（如 X 和 U，或 Y 和 V）才能使 A 与 B 无法通信。任意单一路由器故障，均有冗余路径连通。

> PDF 中该题已有预设答案"2"，经验证正确。

---

## 试题 8：最少路由器间线路故障数
**答案：2**

切断 2 条路由器间线路（如 T-X 与 T-U，或 W-Y 与 W-V）即可使 A 与 B 隔离。因核心网络高度互联，单条链路故障不足以断开连接。

## 试题 9：带宽差距的非合理解释
**答案：B**

- A ✓ 1 Gbps 是最大带宽（霍克尼公式中的 r∞），实际体验可以低得多——合理。
- B ✗ **不合理。** 学生有速度极快的笔记本电脑不会导致体验带宽降低，反而应有帮助。
- C ✓ 光纤只是端到端路径的一部分，路径中其他环节可能成为瓶颈——合理。
- D ✓ 与邻居共享网络交换机会降低可用带宽——合理。
